Connect Four Game

Description

Connect Four is a two-player web-based game developed using HTML, CSS, and JavaScript. Players take turns dropping colored discs into a 7-column, 6-row grid. The objective is to connect four discs horizontally, vertically, or diagonally before the opponent.

Features

- Two-player gameplay
- Turn-based game logic
- Winner detection
- Game restart functionality
- Interactive user interface
- Responsive design

Technologies Used

- HTML
- CSS
- JavaScript

How to Play

1. Open the game in a web browser.
2. Player Red starts the game.
3. Click on a column to drop a disc.
4. Players alternate turns.
5. The first player to connect four discs in a row wins.
6. Click the "Restart Game" button to start a new game.

Project Objective

The objective of this project is to develop problem-solving and web development skills by implementing game logic, user interaction, and dynamic interface design using JavaScript.
